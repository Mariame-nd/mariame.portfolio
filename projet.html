1️⃣ Simulation Réseau avec Cisco

Description du projet :

Ce projet consiste en la mise en place d'une infrastructure réseau simulée avec Cisco Packet Tracer. Il comprend un serveur, un switch et plusieurs PC configurés pour assurer la communication entre les machines. L'objectif est de comprendre et d'appliquer les principes de base de l'administration réseau, notamment la configuration des adresses IP, la segmentation en sous-réseaux et la gestion des protocoles de communication. La simulation permet de tester différentes topologies et d'optimiser la connectivité des équipements. Ce projet illustre l'importance de la configuration réseau dans un environnement informatique et permet de renforcer les compétences en administration réseau et en dépannage des infrastructures connectées.

Technologies utilisées :

  • Cisco Packet Tracer
  • Protocoles réseau : TCP/IP, DHCP, HTTP
  • Configuration CLI (Command Line Interface) pour les équipements réseau

Mon rôle :

Mon rôle est de configurer les adresses IP sur les PC et le serveur et de mettre en place les VLANs pour augmenter le réseau.

Défis et solutions :

  • Régler les problèmes d'interconnexion

    • ✅ Solution : Vérification de la configuration DHCP et activation du service sur le serveur

  • Gestion des VLANs

    • ✅ Solution : Correction des affectations VLAN sur le switch

2️⃣ Portfolio Personnel

Description du projet :

Ce projet consiste à la création d'un portfolio en HTML et CSS permettant de présenter les parcours académiques, les compétences et les réalisations. L'objectif est de concevoir un site moderne, responsive et bien structuré. Le design repose sur une interface épurée avec une navigation fluide, des couleurs harmonieuses et une typographie professionnelle. Chaque section est clairement définie : une page d'accueil accueillante, une présentation détaillée de mon profil, une section dédiée à mes projets et un formulaire de contact. Ce projet met en application les bases du développement web, notamment la structuration des contenus, la gestion des styles et l'optimisation de l'expérience utilisateur sur différents écrans.

Technologies utilisées :

  • HTML
  • CSS
  • Flexbox et Grid
  • Google Fonts

Mon rôle :

Mon rôle est de structurer les différentes sections et améliorer l'interface.

Défis et solutions :

  • Gérer les problèmes d'alignement des sections

    • ✅ Solution : Résolution avec Flexbox et Grid pour une meilleure organisation

  • Mise en place d'un effet survol dynamique

    • ✅ Solution : Utilisation de transitions CSS pour un rendu fluide

3️⃣ Site web pour un restaurant Africain

Description du projet :

Ce projet vise à concevoir un site vitrine pour un restaurant Africain fictif nommé 'Téranga Délices'. Il présente le menu, les avis clients, les spécialités culinaires et les informations de contact. Le site est conçu pour offrir une expérience immersive en mettant en avant des couleurs chaleureuses et des images attractives. Une attention particulière est portée pour garantir une utilisation fluide sur tous les appareils. Le site est également multilingue, permettant aux utilisateurs de basculer entre le français et l'anglais. Ce projet permet de mettre en pratique l'organisation des contenus, la gestion du design responsive et l'intégration d'éléments multimédias.

Technologies utilisées :

  • HTML
  • CSS
  • Flexbox et Grid
  • Google Fonts

Mon rôle :

Mon rôle est de créer les différentes pages et ajouter une navigation fluide avec un menu fixe et des liens internes.

Défis et solutions :

  • Gérer la gestion du multilingue

    • ✅ Solution : Ajouter un bouton de changement de langue

  • Avoir une meilleure aperçu

    • ✅ Solution : Mise en place d'un design responsive
